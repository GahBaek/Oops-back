### 🤝 Branch Naming Convention

| Prefix  | Description                      |
|---------|----------------------------------|
| main    | 실제 서비스가 배포되는 브랜치         |
| develop | 배포 전 전체 개발이 이루어지는 브랜치   |
| feature | 기능 단위 개발 브랜치               |
| hotfix  | 서비스 중 긴급 수정 브랜치            |
